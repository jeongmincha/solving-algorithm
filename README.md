# 알고리즘 문제 및 해결 방법

인터넷에 게재되어 있는 알고리즘 문제들을 저의 방식대로 푼 솔루션들을 모아두었습니다.

# 폴더 구조

```
└ algospot/ (알고스팟)
└ BOJ/ (백준 온라인 저지)
└ codewars/
└ codility/
└ hackerrank/
└ kakao-blind/
└ leetcode/
└ programmers/
```

# Typescript 솔루션 실행 방법
```
# 예를 들어 leetcode/medium/215-kth-largest-element-in-an-array.ts 같은걸 실행시키고 싶다면,
$ npm test -- leetcode/medium/215 # 파일명 전부 다 쓸 필요는 없음.
```
